## 프로젝트 실행 방법
1. 저장소를 클론합니다.
git clone https://github.com/thisisdana49/wanted-pre-onboarding-frontend.git

2. 프로젝트 폴더로 이동합니다.
cd wanted-pre-onboarding-frontend

3. 필요한 종속성을 설치합니다.
npm install

4. 프로젝트를 실행합니다.
npm start

5. 웹 브라우저에서 http://localhost:3000을 열어 프로젝트를 확인합니다.
